Network¶

ntohs(netshort)¶

This subroutine converts values between the host and network byte order. Specifically, ntohs() converts 16-bit quantities from network byte order to host byte order.

Parameters

netshort – 16-bit short addr

Return type

converted addr

Inplace callbacks¶

inplace_cb_reply(qinfo, qstate, rep, rcode, edns, opt_list_out, region, \*\*kwargs)¶

Function prototype for callback functions used in register_inplace_cb_reply, register_inplace_cb_reply_cache, register_inplace_cb_reply_local and register_inplace_cb_reply_servfail.

Parameters

  • qinfo – query_info

  • qstate – module_qstate

  • rep – reply_info

  • rcode – return code (integer), check RCODE_ constants.

  • edns – edns_data

  • opt_list_out – edns_option. EDNS option list to append options to.

  • region – regional

  • **kwargs – Dictionary that may contain parameters added in a future release. Current parameters: repinfo: comm_reply. Reply information for a communication point.

  • msg – string message

log_warn(msg)¶

Parameters

log_hex(msg, data, length)¶

Log a hex-string to the log. Can be any length. performs mallocs to do so, slow. But debug useful.

Parameters

  • msg – string desc to accompany the hexdump.

  • data – data to dump in hex format.

  • length – length of data.
